5. Клонируйте репозиторий на свой компьютер. Перейдите на страницу вашего репозитория на GitHub и нажмите зеленую кнопку "Code". Скопируйте URL, который отображается. Затем откройте терминал на своем компьютере, перейдите в директорию, где вы хотите хранить проект, и введите:

git clone [URL репозитория]

6. Начните работу над проектом. Вы можете изменять файлы, создавать новые и т. д.

Зафиксируйте свои изменения. Когда вы сделаете некоторые изменения, которые вы хотите сохранить, вы должны сделать коммит. В терминале, находясь в директории вашего проекта, выполните следующие команды:

git add .
git commit -m "Your commit message"

Первая команда добавляет все измененные файлы в "staging area", подготавливая их для коммита. Вторая команда делает коммит со всеми файлами, которые вы добавили.

7. Отправьте свои изменения на GitHub. Чтобы отправить ваши коммиты на GitHub, используйте следующую команду:

git push

Продолжайте работу над своим проектом. Повторяйте шаги 6-8, пока не будете готовы считать свой проект завершенным.
